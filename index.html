MODULATE!

A collection of modules and a main patcher for Max/MSP that aims to be the closest approximation to a physical modular synth.

View on GitHub Download .dmg

What is it?

MODULATE! is a collection of modules and a main environment patch for Max 7 on macOS, specifically designed so that users can take advantage of the amazing software that is MAX/MSP without ever having used it before .

Experience and explore the procedures and techniques of modular synthesis by your own without spending loads of $$$ on hardware!

How is it?

A main patcher file serves as a workspace dedicated to the provided library of modules, allowing you to create modules, save presets and map MIDI controls effortlessly.

All the modules are designed to feel and handle like real ones, the user interface is always simple, intuitive and are never too information-dense because, just like the real deal, you have a manual for each of them to get to know all the specifics and details.

Main Patcher

  • Save up to 21 custom-named presets per patch
  • Assign MIDI controls in an intuitive Ableton Live-manner
